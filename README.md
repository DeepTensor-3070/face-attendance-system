# Face Recognition Attendance System

This project is a simple face recognition-based attendance system using Python, OpenCV, and the `face_recognition` library.  
It detects faces in real-time via webcam and marks attendance in a CSV file.

## Features
- Recognizes faces from a stored dataset
- Logs name & time into a CSV file
- Tkinter GUI with Start/Stop buttons and live attendance table
